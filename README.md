rd-socialbutton
===============

# 何するもの？
[jquery.socialbutton.js](http://itra.jp/jquery_socialbutton_plugin/)をAngularJSプロジェクトで使用出来るようにします。

現時点で使用出来るのは以下です。

* tweetボタン
* Facebook Like
* Google+1 ボタン（未確認）
* はてなブックマークボタン

# 使用方法
```html
<!-- tweetボタンなら -->
<div tweetbutton></div>
```
